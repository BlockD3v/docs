## Components

- **Launchpad** — Fair-launch protocol with bonding curve, finalized via Uniswap V3.
- **Trading** — Spot via Uniswap V3 liquidity plus perpetuals with up to 50x leverage.
- **Prediction Markets** — CLOB client powered by Jump Protocol, settled via UMA oracles.
- **APIs** — REST (Q4 2026), WebSocket (live), CLOB (live).
- **Smart Contracts** — Open source, audited by OpenZeppelin, verified on-chain.
