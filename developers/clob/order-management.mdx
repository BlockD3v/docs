## Place order

**Request**

```json
{
  "type": "place_order",
  "data": {
    "marketId": "0x...",
    "outcome": "YES",
    "side": "buy",
    "orderType": "limit",
    "price": "0.65",
    "size": "1000"
  }
}
```

**Response**

```json
{
  "type": "order_confirmed",
  "data": {
    "orderId": "0x...",
    "marketId": "0x...",
    "side": "buy",
    "price": "0.65",
    "size": "1000",
    "status": "open",
    "timestamp": "2025-06-20T12:00:00Z"
  }
}
```

## Cancel order

**Request**

```json
{
  "type": "cancel_order",
  "data": {
    "orderId": "0x..."
  }
}
```

**Response**

```json
{
  "type": "order_cancelled",
  "data": {
    "orderId": "0x...",
    "status": "cancelled",
    "timestamp": "2025-06-20T12:00:00Z"
  }
}
```

## Order status

**Request**

```json
{
  "type": "order_status",
  "data": {
    "orderId": "0x..."
  }
}
```

**Response**

```json
{
  "type": "order_status_response",
  "data": {
    "orderId": "0x...",
    "marketId": "0x...",
    "side": "buy",
    "price": "0.65",
    "size": "1000",
    "filled": "500",
    "remaining": "500",
    "status": "partially_filled",
    "timestamp": "2025-06-20T12:00:00Z"
  }
}
```
